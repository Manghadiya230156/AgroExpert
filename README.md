# AgroExpert 🌱

AI-powered agriculture expert system built using Django, HTML, CSS, JavaScript, and Machine Learning.

## Features

- Crop Recommendation
- Fertilizer Suggestion
- Yield Prediction
- Weather Information
- Soil Test Requests
- Admin Dashboard
- User Authentication

## Technologies Used

- Django
- Python
- HTML5
- CSS3
- JavaScript
- Machine Learning

## Project Structure

```bash
AgroExpert/
│
├── agroexpert_backend/
├── ml_models/
├── static/
├── templates/
└── manage.py
```

## Installation

```bash
git clone https://github.com/Manghadiya230156/AgroExpert.git
cd AgroExpert
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run server:

```bash
python manage.py runserver
```
